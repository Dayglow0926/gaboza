# Project - Gaboza

Figma를 활용하여 디자인 작업부터
HTML, CSS, Javascript 를 활용한 웹페이지 구현까지 진행하는 여행 정보 사이트

## 💻 프로젝트 소개

여러 테마와 핫플레이스의 정보를 제공하는 여행 정보 사이트 입니다.

## 🕰️ 개발 기간

23년 8월 4일 ~ 23년 8월 20일

## ⚙️ 기술 스택

HTML / CSS / JavaScript

라이브러리 : swiper, sweet alert, AOS

프레임워크 : bootstrap

디자인 툴 : Figma, Photoshop
